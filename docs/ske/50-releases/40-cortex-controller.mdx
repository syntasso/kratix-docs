## Overview

The SKE Cortex Controller allows you to automate the process of populating the Cortex catalog with Entities as well as generate Workflows that can be used to provision services backed by Kratix Promises.

The release manifest is available at:

```sh
http://syntasso-enterprise-releases.s3-website.eu-west-2.amazonaws.com/#ske-cortex-controller/
```
