## Overview

The `ske` CLI is the command-line interface to install and manage an instance of
Syntasso Kratix Enterprise (SKE).

The releases are available at:

```
http://syntasso-enterprise-releases.s3-website.eu-west-2.amazonaws.com/#ske-cli/
```

:::info

If you are downloading directly you may need to add the binary to a safe list on your
computer.

If you are using a mac, you can follow
[these instructions](https://support.apple.com/en-gb/guide/mac-help/mchleab3a043/mac).

:::

Check the release notes below for information on each available version.

## Release notes

## [v0.4.1](http://syntasso-enterprise-releases.s3-website.eu-west-2.amazonaws.com/#ske-cli/v0.4.0/) (2025-02-10)

### Features

* `ske upgrade` command will not perform pre and post upgrade checks

## [v0.4.0](http://syntasso-enterprise-releases.s3-website.eu-west-2.amazonaws.com/#ske-cli/v0.4.0/) (2024-12-11)

### Features

* `ske upgrade` command to manage upgrades via the CLI
* `ske get version` command to list details of available upgrades
* Enable Kratix to be deployed without `cert-manager`

## [v0.3.0](http://syntasso-enterprise-releases.s3-website.eu-west-2.amazonaws.com/#ske-cli/v0.3.0/) (2024-10-16)

### Features

* Use `registry.syntasso.io` images instead of `ghcr.io` when installing SKE

## [v0.2.0](http://syntasso-enterprise-releases.s3-website.eu-west-2.amazonaws.com/#ske-cli/v0.2.0/) (2024-10-09)

#### Bug Fixes

* Fix bug preventing correct SKE and SKE-Operator version being returned

### [v0.1.1](http://syntasso-enterprise-releases.s3-website.eu-west-2.amazonaws.com/#ske-cli/v0.1.1/) (2024-09-19)

#### Features

* `get version` shows both deployed and latest available SKE versions
* Add support for `--context`
* SKE can be installed with the `install` command


#### Bug Fixes

* Better description for `--help`
