## Overview

The SKE Destination Helm Chart is a chart for installing Flux on a Kubernetes Destination and optionally installing the K8s Health Agent.

The release is available via the Syntasso Helm Registry https://syntasso.github.io/helm-charts

```
helm repo add syntasso https://syntasso.github.io/helm-charts
helm repo update
helm install ske-destination syntasso/ske-destination -f values.yaml
```

The packaged chart is available at:

```
http://syntasso-enterprise-releases.s3-website.eu-west-2.amazonaws.com/#k8s-health-agent-helm-chart/
```
