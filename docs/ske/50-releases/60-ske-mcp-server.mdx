## Overview

The SKE MCP Server provides a bespoke MCP Server for the users of your SKE built platform. This will support your platform consumers to understand, request, and check on the status of the resources provisioned through Kratix Promises without needing to understand Kratix.
Documentation on how to use this can be found [here](/ske/integrations/mcp).

The release manifest is available at:

```
http://syntasso-enterprise-releases.s3-website.eu-west-2.amazonaws.com/#ske-mcp-server/
```


Check the release notes below for information on each available version.
