title: Install the Backstage integration
description: Installation instructions for the SKE Backstage integration.
sidebar_label: Overview
---

Install the SKE Backstage integration by configuring both sides of the
integration: Backstage must be able to read generated catalogue entities, and
Kratix must be able to write those entities and receive Resource requests.

1. [Configure Backstage](./configuring-backstage) to read generated catalogue
   entities from the state store used by Kratix.
1. [Configure the plugins](./plugins) so Backstage can render SKE entities and
   send Resource requests to Kratix.
1. [Generate Backstage components](./generating-backstage-components) by
   installing the Backstage Controller and enabling Promises for entity
   generation.
