# ske upgrade
upgrade to the next SKE version

## Description
upgrade to the next SKE version

## Usage
```
ske upgrade [--context] [flags]
```

## Examples
```
ske upgrade --context kind-platform
Upgrading SKE from v0.4.0 to v0.5.0

SKE successfully upgraded
```

## Flags
```
--dry-run   Dry run the upgrade
-h, --help      help for upgrade
```
