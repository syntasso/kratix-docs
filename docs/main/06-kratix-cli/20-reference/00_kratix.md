# kratix
The Kratix CLI

## Description
A CLI tool for Kratix

## Usage
```
kratix [command]
```

## Examples
```
# To initialize a new promise
kratix init promise promise-name --group myorg.com --kind Database
```

## Flags
```
-h, --help      help for kratix
-t, --toggle    Help message for toggle
-v, --version   version for kratix
```
